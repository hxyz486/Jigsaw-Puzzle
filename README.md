# 全流程拼图

**注意！！！**

`此版本较旧，拼接小碎片的时候会损害图片`

`所以请把它当成一个纯粹的拼图工具使用`

## 使用方法

1. 在当前路径下打开 cmd，输入：
   ```bash
   pip install -r requirements.txt
   ```

2. 安装完毕后点击 `启动.exe` 运行

## 项目结构

```
Jigsaw-Puzzle/
├── 一键使用.py              # 一键运行脚本
├── 启动.bat / 启动.exe      # 启动入口
├── requirements.txt         # Python 依赖
├── UI/                      # GUI 界面代码
├── gaps-main/               # 核心拼图算法（遗传算法）
└── 114514/                  # 示例拼接结果
```
