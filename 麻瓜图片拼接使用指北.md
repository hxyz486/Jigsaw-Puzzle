## 准备工作

### 1. 安装 Python 依赖

**只需要做一次。** 在项目目录打开终端，运行：

```bash
pip install -r requirements.txt
```

这会自动安装以下必需库：

| 库 | 用途 |
|----|------|
| `customtkinter` | 现代化界面 |
| `pillow` | 图片处理 |
| `numpy` | 数值计算 |
| `matplotlib` | 图表（Gaps 算法用） |
| `opencv-python` | 图像分析（Gaps 用） |
| `click` | 命令行接口（Gaps 用） |

> 如果电脑没有 Python，先去 https://www.python.org/downloads/ 下载安装，安装时勾选 **Add Python to PATH**。

### 2. 素材准备

| 素材 | 说明 |
|------|------|
| 一组分割好的小图片 | 放在 `114514` 文件夹里，支持 jpg / png / bmp / gif |
| 一张完整的原始大图 | 用于生成拼图并还原，什么尺寸都行 |

> 💡 **测试推荐**：项目根目录已自带 `123.png`（831×1296），可以直接用来测试"拼图排序"功能，无需额外准备图片。

### 3. 启动

**最简单：双击 `启动.exe`**，自动打开三合一程序。

> 注：如果 exe 被误删/报毒，备选方案——双击 `启动.bat` 或在终端运行 `python 一键使用.py`。

## 推荐方式：一键使用（三合一）

> **直接双击 `启动.exe`**，无需任何命令行操作。

### 操作流程

1. **双击 `启动.exe`** 启动
2. 点击顶部标签切换功能：**超级拼接** → **正方形转换** → **拼图排序**
3. 每个功能的子步骤通过**左侧导航**切换
4. 在右侧区域完成参数设置和操作

### 三个功能速览

| 标签 | 功能 | 说明 |
|------|------|------|
| **超级拼接** | 多图拼成大图 | 把 `114514` 里的碎片拼成完整大图，输出到 `114514\拼接结果.jpg` |
| **正方形转换** | 拉伸图片 | 按网格拉伸，让每个格子变成正方形，为拼图排序做准备 |
| **拼图排序** | 生成 & 还原拼图 | 生成打乱拼图 → 遗传算法自动还原，输出到 `10086\` |

> ⚠ 拼图排序还原环节，点击"开始还原"后会弹出 CMD 命令提示，推荐复制命令到 CMD 执行以避免 GUI 卡顿。

## 备用方式：独立程序（单功能）

如果你只需要某个单独功能，可以运行 `UI` 文件夹里的程序。

| 文件 | 功能 |
|------|------|
| `UI\超级拼接_CTk.py` | 多图拼接 |
| `UI\正方形转换器_CTk.py` | 网格拉伸 |
| `UI\run_gaps_gui_CTk.py` | 拼图生成与还原 |

```bash
python UI\超级拼接_CTk.py
python UI\正方形转换器_CTk.py
python UI\run_gaps_gui_CTk.py
```

### 超级拼接 — 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 列数 | 0 (自动) | 每行放几张图 |
| 单格宽度 | 200 px | 每张小图缩放宽度 |
| 单格高度 | 200 px | 每张小图缩放高度 |
| 间距 | 0 px | 图与图之间的间隔 |
| 背景颜色 | white | 间隔区域的颜色 |
| 排序 | 名称升序 | 排列顺序（名称/时间） |

> 📁 默认输出：`114514\拼接结果.jpg`

### 正方形转换 — 三种模式

| 模式 | 效果 |
|------|------|
| **面积不变**（默认） | 宽高同时调整 |
| **保持宽度** | 宽度不动，只拉高度 |
| **保持高度** | 高度不动，只拉宽度 |

> 📁 默认输出：`原文件名_square.扩展名`

### 拼图排序 — 参数说明

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 碎片大小 | 32–128 px | 64 | 每个碎片的边长 |
| 进化代数 | 1–9999 | 20 | 迭代轮数，越大越准 |
| 种群大小 | 1–99999 | 200 | 候选解数量，越大越稳 |

> 📁 打乱拼图：`10086\xxx_puzzle.jpg`
> 📁 还原结果：`10086\solved_result.png`

## 常见问题

### Q: 拼接显示"文件夹中没有图片"？
A: 检查图片是否在 `114514` 文件夹，支持 jpg / png / bmp / gif / tiff / webp。

### Q: 正方形拉伸后格子不完美？
A: 像素取整偏差 < 1px，不影响后续拼图。

### Q: 拼图排序提示碎片不能整除图片？
A: 先做正方形转换拉伸图片，或用「智能适配」自动裁剪。

### Q: 还原很慢？
A: 减少进化代数或种群大小，智能适配会自动推荐合理参数。

### Q: 还原结果不对？
A: 增大进化代数和种群大小重试。建议复制命令到 CMD 中运行，避免 GUI 卡顿。

### Q: GUI 还原拼图时无响应？
A: 点击"开始还原"后建议点击「复制命令」，粘贴到 CMD 中运行。Gaps 计算量大，在 GUI 内直接跑可能导致窗口假死。
